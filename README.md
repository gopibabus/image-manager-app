![image-manager-app](./banner.png)

> Simple app built on Vue.js for managing(uploading and viewing) images.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
